# Система распознавания объектов на изображениях и видео

## Описание проекта

Данный проект представляет собой программную систему распознавания объектов на изображениях и видео с использованием модели YOLOv8 и технологий искусственного интеллекта.

Приложение позволяет:

* загружать изображения;
* загружать видеофайлы;
* автоматически обнаруживать объекты;
* отображать результаты распознавания;
* выводить статистику по найденным объектам;
* сохранять обработанные результаты.

Для распознавания используется предварительно обученная модель YOLOv8.

---

* Python 3.10+

## Установка проекта

### 1. Клонирование репозитория

```bash
git clone https://github.com/ВАШ_АККАУНТ/object-detection-system.git
cd object-detection-system
```

---

### 2. Создание виртуального окружения

Windows:

```bash
python -m venv venv
venv\Scripts\activate
```

Linux / macOS:

```bash
python3 -m venv venv
source venv/bin/activate
```

---

### 3. Установка необходимых библиотек

```bash
pip install -r requirements.txt
```

Если файла requirements.txt нет, установить зависимости вручную:

```bash
pip install ultralytics
pip install opencv-python
pip install pyqt6
pip install numpy
```

---

## Загрузка модели YOLOv8

При первом запуске библиотека Ultralytics автоматически скачает модель:

```python
from ultralytics import YOLO

model = YOLO("yolov8n.pt")
```

Если автоматическая загрузка не выполняется, модель можно скачать вручную:

https://github.com/ultralytics/assets/releases

Файл:

```text
yolov8n.pt
```

необходимо поместить в папку:

```text
models/
```

## Запуск программы

Из корневой директории проекта выполнить команду:

```bash
python main.py
```

После запуска откроется графическое окно программы.

---

## Использование программы

### Распознавание изображений

1. Нажать кнопку «Выбрать изображение».
2. Выбрать файл формата:

   * JPG
   * JPEG
   * PNG
3. Дождаться завершения обработки.
4. Просмотреть результат распознавания.

---

### Распознавание видео

1. Нажать кнопку «Выбрать видео».
2. Выбрать файл формата:

   * MP4
   * AVI
   * MOV
3. Запустить обработку.
4. Дождаться окончания анализа видео.
5. Просмотреть сохранённый результат.

---

## Пример работы

После обработки изображения система отображает:

* ограничивающие рамки объектов;
* название объекта;
* статистику обнаруженных объектов.

Пример:

```text
Человек — 3
Автомобиль — 2
Велосипед — 1
```

## Возможные проблемы

### Ошибка ModuleNotFoundError

Установить отсутствующую библиотеку:

```bash
pip install название_библиотеки
```

---

### Ошибка при загрузке модели

Проверить наличие файла:

```text
yolov8n.pt
```

или подключение к интернету для автоматической загрузки модели.

---

### Медленная обработка видео

Скорость зависит от:

* мощности процессора;
* объёма оперативной памяти;
* размера видеофайла;
* наличия видеокарты.
